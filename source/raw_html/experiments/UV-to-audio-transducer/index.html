UV Sound Converter

Transform Sunlight Into Sound

What Is It?

This DIY project converts UVA and UVB sunlight into sound using sensors and a microcontroller. It's perfect for experimental artists, educators, or sound designers interested in environmental interactivity.

How It Works

  • UV sensors detect UVA and UVB radiation
  • Microcontroller reads and maps the data to sound parameters
  • Audio output is generated based on light intensity and variation

Components Needed

  • VEML6075 UV Sensor – Detects UVA & UVB
  • Arduino/Teensy – Processes the data
  • Audio Adapter or MIDI Output – Converts data to sound
  • Speaker or Synth – Outputs the audio
  • Power Supply – USB battery or solar panel

Optional Features

  • Stereo panning based on dual sensor input
  • Use cloud cover or UV fluctuation to trigger rhythm
  • Control a DAW via MIDI for advanced soundscapes

Getting Started

Hook up your VEML6075 sensor to an Arduino or Teensy. Use the UV readings to drive pitch, volume, or rhythm. Check out this GitHub repo for sample code and schematics.

Use Cases

  • Ambient sound installations
  • Weather-reactive music generators
  • Interactive outdoor sculptures
